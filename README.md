# Vue Contacts

A Quasar Project for browsering contacts

## Prerequisite
+ "node": ">= 10.18.1" and "< 17"
+ "npm": ">= 6.13.4"


### Install the dependencies
```bash
npm install
```

### Run (Start the app in development mode)
```bash
npm run dev
```
Then enjoy by entering http://localhost:8080/#/
